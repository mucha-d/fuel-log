## Instalacja projektu po sklonowaniu repozytorium

### 1. Sklonuj repozytorium
git clone https://github.com/mucha-d/Refueling-App.git

### 2. Zainstaluj zależności Node
npm install

### 3. Zbuduj frontend (Ionic React → tworzy build/)
ionic build

### 4. Zsynchronizuj projekt z Capacitor
npx cap sync

### 5. Otwórz projekt Android w Android Studio
npx cap open android
